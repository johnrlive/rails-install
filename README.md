# rails-install
Bash Script To Ruby, Nginx, Passenger and Postgres on a fresh Ubuntu install

# How To Install:

1. sudo apt-get install -y git
2. cd ~
2. git clone https://github.com/johnrlive/rails-install.git
3. cd rails-install
4. bash install.sh
